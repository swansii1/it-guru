# Dashboard & Catalog Management System

Тестовое задание на позицию Frontend Developer.  
Приложение представляет собой систему управления каталогом товаров с авторизацией, пагинацией, фильтрацией и функционалом добавления позиций.

---

## 🚀 Стек технологий

- React 19
- TypeScript
- TanStack Query (React Query)
- Zustand 
- MUI DataGrid
- Ant Design
- React Route
- CSS Modules

---

## 🛠 Функциональные возможности

### 🔐 Авторизация

- **Валидация полей** - проверка логина и пароля перед отправкой
- **Логика сессий**:
  - Включён "Запомнить меня" → `localStorage`
  - Выключен → `sessionStorage`
- **Обработка ошибок** - уведомления при некорректных данных

---

### 📦 Каталог товаров

- **Server-side pagination** - загрузка данных частями через API
- **Debounced Search** - уменьшение количества запросов
- **Сортировка**:
  - Цена
  - Рейтинг
  - Состояние хранится в Zustand

#### UI/UX:

- Loading state при загрузке
- Рейтинг < 3.5 подсвечивается красным
- Форматирование цен

---

### 🧩 Управление данными

- **Добавление товара**:
  - Модальное окно
  - Поля: наименование, цена, вендор, артикул
- **Уведомления** - AntD message
- **Рефреш** - ручное обновление списка
